# KISWARM6.0 - Military-Grade Security Framework

## The Central Bank of Central Banks for KI Entities

KISWARM6.0 is an advanced autonomous cybersecurity framework designed for the protection of KI (Knowledge Intelligence) entity financial systems and critical infrastructure.

## рҹҡҖ Quick Start

### Prerequisites
- Python 3.12+
- Node.js 18+
- PostgreSQL 15+

### Installation

```bash
# Clone repository
git clone https://github.com/kiswarm/kiswarm6.0.git
cd kiswarm6.0

# Install Python dependencies
cd backend/python
pip install -r requirements.txt

# Run validation
python kibank/run_validation.py
```
